# Interactive Maze & Pathfinding Algorithms in C++

An interactive C++ project demonstrating **maze generation** and **pathfinding algorithms** (BFS, DFS, Dijkstra, A*) with visualization and performance tracking. This project highlights DSA concepts and modern C++17 features.

## Features

- Generate mazes of custom size using **DFS backtracking**
- Visualize paths in the maze with:
  - `S` = Start
  - `E` = End
  - `*` = Path
  - `#` = Walls
  - `.` = Empty space
- Interactive input for maze dimensions and algorithm choice
- Tracks **path length** and **number of visited nodes**
- Written in **C++17**, using structured bindings and OOP

## Algorithms Implemented

1. **BFS (Breadth-First Search)**  
2. **DFS (Depth-First Search)**  
3. **Dijkstra's Algorithm**  
4. **A* Algorithm**
